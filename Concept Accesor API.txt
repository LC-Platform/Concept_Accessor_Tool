Concept Accessor API Docs
Base URL: http://10.2.8.12:3000
API Endpoints


1. PDF Processing
Upload and Process PDF
text
POST /read-pdf/
Content-Type: multipart/form-data
Body: pdf (file)


Response:
{
  "chapter_id": "chapter_a1b2c3d4",
  "section_ids": ["3.1", "3.2", "3.1.1"],
  "message": "Processed & stored"
}

2. Summary Generation
Generate Full Chapter Summary
text
POST /full-summary/
Content-Type: application/json
Body: {"chapter_id": "chapter_a1b2c3d4"}


Response:
{
  "chapter_id": "chapter_a1b2c3d4",
  "full_summary": "Generated summary text..."
}

Generate Section Summary
text
POST /section-summary/
Content-Type: application/json
Body: {
  "chapter_id": "chapter_a1b2c3d4",

}


Response:

  "chapter_id": "chapter_a1b2c3d4",
  "section_id": "3.1",
  "section_summary": "Section summary text..."
}

5. Audio Services
Get Domain Term Audio
text
POST /domain-term-audio/
Content-Type: application/json
Body: {
  "domain_id": "bryophyte",
  "chapter_id": "chapter_a1b2c3d4"
}


Response: Audio file stream (MP3 format)
Content-Type: audio/mpeg
